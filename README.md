# Lawbric – AI Built for Law Firms

A modern, professional one-page marketing site for **Lawbric**, an AI platform for law firms (voice receptionists, chatbots, client intake automation, appointment scheduling, workflow automation).

## Stack

- **React 18** + **Vite**
- **Tailwind CSS 3** (custom colors: Lawbric blue `#0284c7`, grays)
- Sticky header, smooth scroll, scroll-triggered fade-in animations, mobile responsive

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build for production

```bash
npm run build
npm run preview
```

## Sections

1. **Hero** – Headline, subheadline, primary/secondary CTAs  
2. **Problem** – “Law firms lose leads after hours” + pain points  
3. **Solution** – Meet Lawbric AI (4 feature cards)  
4. **How It Works** – 3-step process  
5. **Why Lawbric** – Built for law firms (bullets)  
6. **Testimonials** – 3 quote cards  
7. **Final CTA** – “See the AI receptionist in action”  
8. **Footer** – Logo, links, © 2026

Design: clean typography (DM Sans), soft shadows, blue/gray palette, law-firm appropriate.
